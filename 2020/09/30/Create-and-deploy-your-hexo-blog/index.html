Create and deploy your hexo blog

Set up

  1. Install Node.js
  2. Install Git
  3. Create a Github account

Once you finished installing, open the command prompt and install hexo by typing the following command.

npm install -g hexo-cli

Move to your working directory and initiate your hexo blog

hexo init

Hexo automatically generates these files

Test it by running

hexo server

Open your browser and go to port http://localhost:4000/ it should give you a page that looks this.

Create your first blog

Create your first blog by running

 hexo new "blog_title"

In my case i’m doing

hexo new "Create a Blog and deploy it with Hexo"

you can view your blog files under

.\source\_posts

All of them are in markdown format (.md)

  • https://www.markdowntutorial.com/

Edit them by using any text-editor you prefer. Here are some free online editors

  • https://stackedit.io/
  • https://dillinger.io/
  • https://pandao.github.io/editor.md/en.html

If you wanna add some images to your blog, you could upload your image file to a image hosting web site first and paste the link in your .md file.Here are some free image hosting web sites

  • https://imgchr.com/?lang=en
  • https://imgur.com/
  • https://imgbb.com/

Deploy your blog on github page

Once you’ve done editting your blog, i can deploy it
Sign in to your github account, Create a new repository.
The repository name should be in this format

 [account_name].github.io

in my case:

Next we’ll install a plug-in that deploys our blog to github, run:

npm install --save hexo-deployer-git

After that we need to add a few line to the configuration file.
open: _config.yml

Scroll all the way down to the bottom, under Deployment type this

Repo Address can be found here, make sure you choose SSH

To deploy your website enter the following command in your command propmt.

hexo clean                        ///clean old static files
hexo generate                     ///generate new static files
hexo deploy                       ///deploy

Step 1: Check your SSH key

Step 2 : Generate new SSH key in git bash

Enter the path you want your ssh to be saved

You can enter the passphrase if you want to, but you don’t have to.
If you enter the passphrase it’s gonna ask you for it each time you update it.

  1. Open the new generated “id_rsa.pub” in my case it’s under this path c/Users/sean/.ssh/id_rsa.pub, CTRL-A select all and copy it.
  2. login to github,go to Settings –> SSH and GPG keys –> New SSH key，add SSH key
  3. paste the key in the box, title is optional.
